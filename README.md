# Heracles Path ⚡

Mitoloji + Adler Psikolojisi + AI

## Deploy

1. Bu repoyu GitHub'a yükle
2. Netlify'da "Import from GitHub" ile bağla
3. Netlify → Site settings → Environment variables → `GROQ_API_KEY` ekle
4. Bitti.

## Lokal geliştirme

```bash
npm install
npm run dev
```
